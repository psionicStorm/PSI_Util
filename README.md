##PSI##
###Cuckoo Hashing###

__Install:__
```
sudo make install
```
__Clean:__ 
```
sudo make clean
```

__Remove:__ 
```
sudo make remove
```

__Dependencies:__ libglib2.0-dev, libssl-dev

__Usage:__ the library can be included using -lpsi-util option and 
headers using -I/usr/include/psi_util option.
