# Private Set Intersection (PSI)
###Cuckoo Hashing###
---
__Install:__
```
sudo make install
```
__Clean:__ 
```
sudo make clean
```

__Remove:__ 
```
sudo make remove
```

###Dependencies:
---
 * libglib2.0-dev
 * libssl-dev

###Usage:
---
This library can be included using -lpsi-util option and 
headers using -I/usr/include/psi_util option.
